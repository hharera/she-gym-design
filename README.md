# She-Gym — Design Preview

Static, single-file design preview for the She-Gym mobile app.

Open `index.html` in any browser — or view the published version on GitHub Pages.

## What's inside

- **Design tokens** as CSS variables (colors, typography, spacing, radius, elevation, motion)
- **Light + dark themes** with a top-right toggle
- **Component gallery** (buttons, inputs, cards, list tiles, chips, empty state)
- **Five representative screens** in mobile phone frames:
  - Welcome
  - Home / Dashboard
  - Class schedule
  - Class detail / booking
  - Profile

## Design language

Claude-inspired — warm cream surfaces, soft rounded geometry, editorial serif headlines, restrained violet accent, generous whitespace. She-Gym brand identity (logo, violet accent) preserved.
